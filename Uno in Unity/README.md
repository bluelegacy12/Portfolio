# Uno-Game-in-Unity
My first crack at C# and Unity. It is a simple Uno card game.

Use the .zip file for the desktop version (just extract the zip file and click the .exe file within it) 
and for androids, the .apk file can be downloaded directly to your android device.

You will need to download the whole .zip folder and the Uno 3D.exe app should pull up the game. The screen will be forced full screen by default because the game needs 1920x1080 resolution for the UI to line up properly. If your internal resolution is anything different, then it may not line up well. Sorry!
EDIT: The game now supports more resolutions! Yay! Smaller resolutions than 1080p will still start to get claustrophobic though, and even smaller res will render the game unplayable. But higher res should be fine!

UPDATE: The game supports up to four players! But you have to crowd around the keyboard.

NOTE: This game is not in 3D it just uses a 3D core cause that's what I was following along with in my research to getting started in Unity.
